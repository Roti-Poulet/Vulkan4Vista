# Vulkan4Vista
A patch for Vulkan for Windows Vista

WARNING: This patch is very experimental, vulkan programs probably won't work!!!

**Requirements:**

-Vista 64 bit SP2. I won't make 32 bit DLLs

-Modded Nvidia GPU drivers (47x.xx), follow this guide: https://board.eclipse.cx/viewtopic.php?t=859

-Preferably Nvidia Maxwell 2.0 or newer GPU. I haven't tested on older ones

**Features:**

-Semi bypass of the "-3" error

-Working basic Vulkan API calls

-Based on the driver 472.12: Theoretical Vulkan 1.2.175 support
